# CatHeadTab 隐私政策

**最后更新日期：2026年5月11日**

CatHeadTab 是一款浏览器新标签页扩展程序，用于提供可自定义桌面、小组件、可选云同步和可选 AI 功能。本隐私政策说明 CatHeadTab 会处理哪些数据、哪些情况下数据会离开您的设备，以及可能与哪些相关方共享。

## 摘要

默认情况下，CatHeadTab 的大部分数据都在您的设备本地处理。某些可选功能或联网功能需要传输有限数据才能工作，包括账号登录、云同步、天气、金融、热点内容、网站图标获取和服务端 AI。CatHeadTab 不会出售、出租、交易您的数据，也不会将您的数据用于广告或跟踪。

## 本地数据

CatHeadTab 会使用 `chrome.storage.local`、IndexedDB 或浏览器本地存储保存以下本地数据：

| 数据 | 用途 | 本地存储位置 |
|------|------|--------------|
| 桌面布局、图标、文件夹、小组件、小组件设置和便签内容 | 渲染和自定义新标签页 | `chrome.storage.local` |
| 用户偏好、语言、壁纸设置、锁屏超时和后端服务器地址 | 持久化设置 | `chrome.storage.local` |
| AI 提供商配置和用户填写的 AI API 密钥 | 连接用户选择的本地 AI 提供商 | `chrome.storage.local` |
| AI 聊天记录 | 方便保留最近对话 | `chrome.storage.local` |
| 自定义壁纸图片和网站图标缓存 | 高效显示资源 | IndexedDB |
| 天气、股票和汇率缓存 | 减少重复网络请求 | 浏览器本地存储 |

您在扩展中填写的 AI API 密钥只保存在本地，不会发送到 CatHeadTab 服务器。

## 可选账号和云同步

如果您选择创建账号、登录账号，或将扩展连接到后端服务器，CatHeadTab 可能会将以下数据传输并存储到官方 CatHeadTab 后端，或您配置的自托管后端：

- 账号数据，例如电子邮件地址、用户名、密码哈希、邮箱验证状态、头像和会话令牌。
- GitHub 或 Google OAuth 数据，例如提供商 ID、邮箱地址、用户名、头像 URL 和 OAuth 令牌。
- 桌面布局、小组件设置、用户偏好、锁屏超时、壁纸 URL 和自定义壁纸图片。
- 服务端 AI 限额所需的用量元数据，例如请求次数和 token 数量。

云同步是可选功能。当前扩展中，浏览器书签和浏览历史通过 Chrome API 在本地显示，不会通过云同步上传。

## 浏览器 API 访问权限

CatHeadTab 为用户可见功能请求以下 Chrome 扩展权限：

| 权限 | 用途 |
|------|------|
| `storage` | 在本地保存布局、设置、AI 配置和聊天记录 |
| `unlimitedStorage` | 存储壁纸和网站图标资源，避免普通存储额度限制 |
| `bookmarks` | 在书签浏览器和本地 AI 模式中显示、搜索您的浏览器书签 |
| `history` | 在历史记录浏览器和本地 AI 模式中显示、搜索您的浏览历史 |
| `favicon` | 在快捷方式、书签和历史记录旁显示浏览器提供的网站图标 |
| `activeTab` | 当您使用“添加到桌面”弹出窗口时读取当前标签页 URL 和标题 |
| `system.cpu` | 在系统监控小组件中显示 CPU 使用情况 |
| `system.memory` | 在系统监控小组件中显示内存使用情况 |
| `declarativeNetRequest` | 为新浪财经股票行情请求添加所需的 `Referer` 请求头 |
| `search` | 当您从新标签页提交网页搜索时，通过浏览器默认搜索服务发起搜索 |

## 主机权限

CatHeadTab 请求 `http://*/*` 和 `https://*/*` 主机权限。这些较宽的主机权限仅用于已经实现的用户可见功能：

| 范围 | 用途 |
|------|------|
| 您通过 CatHeadTab 添加或打开的网站 | 为桌面快捷方式获取网页标题和网站图标 |
| 用户配置的本地 AI 端点 | 允许扩展连接到用户选择的 OpenAI 兼容或自托管 AI API |
| AI 提供商 API | 将本地 AI 请求直接发送到您配置的提供商，例如 OpenAI、DeepSeek、Google Gemini、Anthropic、Aliyun/Qwen、Zhipu、Moonshot、Minimax 或自托管端点 |
| 金融 API | 在扩展环境中按需直接获取股票行情，例如 Yahoo Finance 和新浪财经 |
| CatHeadTab 或自托管后端 | 提供账号、同步、天气、金融代理、热点、网站图标代理、壁纸和服务端 AI 功能 |

本地 AI 请求可能会通过扩展的后台 Service Worker 中转，以绕过浏览器 CORS 限制。该中转仅发生在已安装的扩展内部，不会将您的本地 AI API 密钥或本地 AI 聊天内容发送到 CatHeadTab 服务器。

## 联网功能

根据您使用的功能，CatHeadTab 可能会联系以下服务：

- **CatHeadTab 后端或自托管后端**：账号登录、云同步、个人资料/头像、自定义壁纸同步、服务端 AI、天气查询、金融代理、热点数据、网站图标代理和壁纸搜索。
- **AI 提供商**：本地 AI 模式会将您的提示词、选定上下文和本地 AI API 密钥直接发送到您配置的 AI 提供商。服务端 AI 模式会将聊天消息发送到已配置的后端，再由该后端转发给服务端配置的 AI 提供商。
- **GitHub 和 Google OAuth**：账号登录和账号绑定。
- **天气和地理编码服务**：基于您配置的城市或后端 IP 到地区查询提供天气。
- **金融数据源**：股票代码、自选股或货币对可能会发送到 Yahoo Finance、新浪财经、Frankfurter 或配置的后端代理。
- **热点数据源**：GitHub、Bilibili、微博、小红书和 BBC News 可能会通过后端请求，用于显示热门内容。
- **网站图标来源和目标网站**：您添加为快捷方式的域名可能会被直接请求，或通过后端图标代理请求。后端可能使用 Google S2、DuckDuckGo 图标、Favicone 或目标网站获取图标。
- **Wallhaven 和配置的壁纸来源**：壁纸浏览和搜索。

## AI 助手

CatHeadTab 支持两种 AI 模式：

- **本地 AI 模式**：您提供 AI API 密钥和端点。您的提示词、本地聊天上下文以及您请求使用的本地数据，会从扩展直接发送到您配置的 AI 提供商。当您明确要求 AI 搜索书签或历史记录时，只有相关工具结果会发送给该 AI 提供商，不会发送到 CatHeadTab 服务器。
- **服务端 AI 模式**：如果连接的后端配置了服务端 AI，您的聊天消息会发送到该后端，并由后端转发给后端配置的 AI 提供商。CatHeadTab 会记录请求次数和 token 数量等用量元数据用于限额控制。CatHeadTab 不会在服务器上把聊天正文保存为 AI 聊天历史。

在服务端 AI 模式下，CatHeadTab 不会向 AI 助手提供书签或浏览历史工具。涉及浏览器书签或浏览历史的请求需要使用本地 AI 模式。

## 数据收集、处理、存储和共享

### 传输到 CatHeadTab 或配置后端的数据

CatHeadTab 只会在启用相关功能且功能需要时将数据传输到后端：

- 登录、OAuth、邮箱验证、密码重置和会话所需的账号与身份验证数据。
- 用于云同步的布局、小组件、偏好、头像、自定义壁纸和相关用户提供内容。
- 天气请求数据，包括城市、语言、单位，以及后端用于自动定位的临时 IP 处理。
- 返回所请求内容所需的金融、热点、网站图标和壁纸请求参数。
- 使用服务端 AI 模式时的服务端 AI 聊天请求和 AI 用量元数据。

### 与第三方共享的数据

CatHeadTab 仅在提供用户可见功能、遵守法律或保护安全所需时共享数据：

- 与您选择的 AI 提供商或后端配置的 AI 提供商共享提示词和必要上下文。
- 当您使用 GitHub 或 Google 登录时，与 GitHub 或 Google 共享 OAuth 登录数据。
- 与相关功能所需的外部服务共享域名、股票代码、货币对、城市名或壁纸搜索参数。
- 如果法律严格要求，向法律机构提供所需数据。

CatHeadTab 不会为了个性化广告、重定向广告、信用评估或数据经纪目的转移、出售、出租或使用用户数据。

## Chrome Web Store Limited Use 声明

CatHeadTab 对来自 Chrome 扩展 API 和 Chrome 权限的信息的使用遵守 Chrome Web Store 用户数据政策，包括 Limited Use 要求。通过 Chrome API 获取的数据仅用于提供或改进 CatHeadTab 的单一目的和用户可见功能：新标签页自定义、书签和历史浏览、快捷方式创建、小组件、可选同步和可选 AI 辅助。CatHeadTab 不会将这些数据用于广告，不会出售这些数据，也不会允许人工读取这些数据，除非获得用户明确同意、出于安全目的、为了遵守法律，或在政策允许的聚合匿名内部运营场景中使用。

## 安全

通过网络传输的数据会在目标服务支持时使用 HTTPS。身份验证信息会受到保护，不会公开披露。本地存储的安全性取决于您的浏览器个人资料和设备安全。

## 自托管

CatHeadTab 支持自托管。如果您配置自己的后端，发送到该后端的数据由您的部署及其运营者控制。

## 儿童隐私

CatHeadTab 不面向 13 岁以下儿童。我们不会有意收集儿童的个人信息。

我们可能会不时更新本隐私政策。更改将发布在此页面上，并更新“最后更新日期”。
