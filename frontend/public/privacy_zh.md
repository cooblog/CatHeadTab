# CatHeadTab 隐私政策

**最后更新日期：2026年4月12日**

CatHeadTab 是一款浏览器新标签页扩展程序，它将您默认的新标签页替换为可自定义、高颜值的桌面界面。本隐私政策解释了我们如何处理您的数据。

## 摘要

**CatHeadTab 不会收集、传输或出售任何个人数据。** 除非您明确选择启用云同步，否则您的所有数据都将保留在您的设备上。

## 数据存储

### 本地数据（仅存储在您的设备上）

| 数据 | 用途 | 存储位置 |
|------|---------|-----------------|
| 桌面布局（图标、文件夹、小组件） | 显示您自定义的新标签页 | `chrome.storage.local` |
| 用户偏好设置（语言、壁纸 URL、锁屏超时） | 持久化您的设置 | `chrome.storage.local` |
| AI 提供商配置和 API 密钥 | 连接到您选择的 AI 服务 | `chrome.storage.local` |
| 自定义壁纸图片 | 显示您上传的壁纸 | IndexedDB |
| AI 聊天记录 | 持久化与 AI 助手的对话 | `chrome.storage.local` |

**重要提示：** AI API 密钥仅存储在您的本地设备上，**绝不会**传输到我们的服务器。

### 可选的云同步（仅限用户主动发起）

如果您选择创建一个帐户并连接到后端服务器（无论是我们的公共服务还是您自己托管的实例），以下数据可能会被同步：

- 桌面布局（页面、程序坞、文件夹、图标）
- 用户偏好设置（壁纸 URL、锁屏设置）
- 自定义壁纸图片（压缩的 WebP 格式）
- 书签（仅用于云端备份）

云同步是**完全可选的**，并且**默认处于禁用状态**。您始终可以控制何时同步以及同步哪些内容。

## 浏览器 API 访问权限

CatHeadTab 请求以下浏览器权限，每项权限仅用于所述目的：

| 权限 | 用途 |
|-----------|---------|
| `storage` | 在本地保存您的布局、设置和偏好 |
| `unlimitedStorage` | 存储壁纸图片而不受存储限制 |
| `bookmarks` | 在书签浏览器中显示和搜索您的书签 |
| `history` | 在历史记录浏览器中显示和搜索您的浏览历史 |
| `favicon` | 在您的桌面快捷方式旁边显示网站图标 |
| `activeTab` | 使用“添加到桌面”弹出窗口时读取当前标签页的 URL 和标题 |
| `system.cpu` | 在系统监控小组件中显示实时 CPU 使用率 |
| `system.memory` | 在系统监控小组件中显示实时内存使用率 |
| `declarativeNetRequest` | 修改新浪财经 API 请求的 `Referer` 标头（获取中国 A 股股票数据所需） |
| `search` | 使用浏览器默认搜索引擎进行网页搜索 |

### 主机权限 (Host Permissions)

为了提供完整的功能，CatHeadTab 请求了对所有网站 (`http://*/*` 和 `https://*/*`) 的主机权限。这主要用于以下用途：

| 主机/范围 | 用途 |
|------|---------|
| **所有网站** (`*://*/*`) | **获取网站图标 (Favicon)**：当您在桌面上添加网站快捷方式时，扩展程序可能会请求该网站以获取其图标，从而提升您的视觉体验。 |
| **所有网站** (`*://*/*`) | **AI API 请求**：允许扩展程序直接从您的浏览器连接到您配置的任何第三方 AI 提供商的 API 端点。 |
| `https://query1.finance.yahoo.com/*` | 获取美国和香港股票市场数据 |
| `https://query2.finance.yahoo.com/*` | 获取美国和香港股票市场数据 |
| `https://hq.sinajs.cn/*` | 获取中国 A 股股票市场数据 |
| `https://api.frankfurter.dev/*` | 获取货币汇率数据（欧洲中央银行） |

这些请求是**直接从您的浏览器**发送到各自的 API 的。CatHeadTab 不会在我们的服务器上代理、记录或存储任何此类金融数据。

## AI 助手

AI 代理功能允许您与大型语言模型 (LLM) 交互以管理您的桌面。使用此功能时：

- **您的 AI API 密钥**存储在您的本地设备上，并直接发送到您选择的 AI 提供商（例如 OpenAI、DeepSeek、Google）。它**绝不会**发送到 CatHeadTab 服务器。
- **聊天消息**直接从您的浏览器发送到 AI 提供商的 API 端点。CatHeadTab 不会在任何服务器上拦截、记录或存储这些消息。
- **聊天记录**保存在您的本地设备上以方便使用，并且可以随时清除。
- AI 助手可以读取您的桌面布局、书签和浏览历史记录，**前提是您要求它这样做**，并且仅用于满足您的请求。**前提是在本地运行AI大模型，我们绝对不会让您上传和保存您的书签和浏览历史记录。**

## 第三方服务

根据您的使用情况，CatHeadTab 可能会与以下第三方服务交互：

- **AI 提供商**（OpenAI、DeepSeek、Google、Anthropic 等）：仅当您配置并使用 AI 助手时。受各提供商自己的隐私政策约束。
- **Wallhaven** (wallhaven.cc)：仅当您浏览在线壁纸时。受 Wallhaven 的隐私政策约束。
- **Yahoo Finance / 新浪财经**：仅当您使用股票追踪小组件时。
- **Frankfurter API**：仅当您使用汇率小组件时。
- **GitHub / Google OAuth**：仅当您选择使用这些服务登录时。

## 数据收集、使用和共享

为了遵守隐私法规和应用商店政策，我们明确披露如何处理您的数据：

### 1. 我们收集哪些数据
- **本地数据**：我们使用 `chrome.storage.local` 将您的桌面布局、用户偏好设置和 AI 聊天记录本地存储在您的设备上。我们还使用 IndexedDB 在本地存储自定义壁纸图片。
- **云同步数据（可选）**：如果您明确创建帐户并启用云同步，我们会在我们的服务器（或您自托管的服务器）上收集并存储您的桌面布局、用户偏好设置、自定义壁纸图片和书签，其唯一目的是在您的设备之间进行同步。
- **身份验证数据**：如果您使用 GitHub 或 Google OAuth 登录，我们会收集您的电子邮件地址和 OAuth 提供商提供的基本个人资料信息，以创建和管理您的帐户。

### 2. 我们如何使用您的数据
- **核心功能**：您的本地数据专门用于渲染您自定义的新标签页并提供扩展程序的核心功能。
- **云同步**：您同步的数据仅用于在您登录的不同设备上恢复您的布局和偏好设置。
- **AI 助手**：如果您使用 AI 助手，您的提示词和相关上下文（如书签或历史记录，仅在您明确请求时）将发送到您配置的 AI 提供商以生成响应。

### 3. 我们如何共享您的数据
- **我们不会向任何第三方出售、出租或交易您的个人数据。**
- **我们不会将您的数据用于广告或跟踪目的。**
- **第三方服务提供商**：
  - **AI 提供商**：如果您配置并使用 AI 助手，您的提示词和必要的上下文将直接与您选择的 AI 提供商（例如 OpenAI、Google、DeepSeek）共享以生成响应。此数据受相应提供商的隐私政策约束。
  - **OAuth 提供商**：如果您使用 GitHub 或 Google 登录，我们会与这些提供商交互以对您进行身份验证。
- **法律合规**：如果法律要求或响应公共机构的有效请求，我们可能会披露您的数据。

## 自托管

CatHeadTab 支持完全自托管。您可以将后端服务器部署在您自己的硬件（Homelab、NAS、VPS）上，将所有同步数据完全置于您的控制之下。

## 儿童隐私

CatHeadTab 不面向 13 岁以下的儿童。我们不会有意收集儿童的个人信息。

## 本政策的变更

我们可能会不时更新本隐私政策。更改将发布在此页面上，并更新“最后更新日期”。
